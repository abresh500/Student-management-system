 student record management system

overview

this project is a simple java program used to manage student records using file i/o. it stores data in files so information is saved even after the program closes.

## system design

* student class: stores id, name, department, and gpa
* studentmanager: handles add, search, update, delete, and display
* report: calculates total students, highest, lowest, and average gpa
* fileutil: shows file info and creates backup

## file handling

the system uses text files, binary files, and object serialization to store data. it also creates files automatically and handles errors using try-catch.

## conclusion

this project shows how java oop and file handling can be used to build a simple data management system.
